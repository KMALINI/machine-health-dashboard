# Acoustic Fault Detection System

A real-time noise monitoring web application that detects sound anomalies 
and displays safety levels through a colour-coded GUI (green/yellow/red). 
Built with TypeScript and React, featuring a GPT-powered chatbot for user 
assistance during detected anomaly events. Developed as part of INDCON'26 
Industrial Innovation Challenge — Silver Prize, Anna University Symposium.

## Tech Stack
- TypeScript
- React
- Tailwind CSS
- GPT API

## How to Run
```sh
git clone <YOUR_GIT_URL>
cd fusion-flow-app
npm i
npm run dev
```
